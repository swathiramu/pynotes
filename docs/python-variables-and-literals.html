Python Variables And Literals

Mon 12 January 2026

Declaring and Assigning Variables

count = 10
price = 99.99
name = "Python"

print(count) 
print(price)   
print(name)    

10
99.99
Python

Multiple Variable Assignment

a, b, c = 1, 2, 3
print(a)  
print(b)  
print(c)

1
2
3

Reassigning Variables

value = 100
print(value)

100

value = "Reassigned"
print(value)

Reassigned

Numeric Literals

integer = 42
print(integer)  

42

float = 3.14
print(float) 

3.14

complex = 2 + 3j
print(complex)

(2+3j)

String Literals

single_quote = 'Hello'
print(single_quote)  

Hello

double_quote = "World"
print(double_quote) 

World

multi_line = """This is a multi-line string"""
print(multi_line)

This is a multi-line string

Boolean Literals

is_active = True
is_logged_in = False

print(is_active)    
print(is_logged_in)

True
False

Special Literal - None

result = None
if result is None:
    print("No value assigned yet")

No value assigned yet

Collection Literals

list_literal = [1, 2, 3]
print(list_literal)

[1, 2, 3]

tuple_literal = (4, 5, 6)
print(tuple_literal)

(4, 5, 6)

set_literal = {7, 8, 9}
print(set_literal)

{8, 9, 7}

dict_literal = {"a": 1, "b": 2}
print(dict_literal)   

{'a': 1, 'b': 2}

Binary, Octal, and Hexadecimal Literals

binary = 0b1010
print(binary)

10

octal = 0o12
print(octal)

10

hexadecimal = 0xA
print(hexadecimal)  

10

Using Variables in Expressions

x = 5
y = 3

sum_result = x + y
product_result = x * y
print(sum_result) 

8

print(product_result)

15
