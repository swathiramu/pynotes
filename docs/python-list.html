Accessing List Elements (Indexing)

f = ["apple", "banana", "orange"] print(f[0])

print(f[-1])

orange

print(f[-2])

banana

print(f[2])

orange

Slicing Lists

num = [0, 1, 2, 3, 4, 5]
print(num[1:4])

[1, 2, 3]

print(num[:3])

[0, 1, 2]

print(num[3:]) 

[3, 4, 5]

Modifying List Elements

A = ["pen", "pencil", "eraser"]
A[1] = "marker"

print(A) 

['pen', 'marker', 'eraser']

A = ["pen", "pencil", "eraser"]
A[2] = "scale"

print(A) 

['pen', 'pencil', 'scale']

Adding Elements to a List

num = [1, 2, 3]
num.append(4)
print(num)

[1, 2, 3, 4]

num = [1, 2, 3]
num.insert(1, 10)
print(num)

[1, 10, 2, 3]

Removing Elements from a List

data = [10, 20, 30, 40]
data.remove(20)
print(data)

[10, 30, 40]

data = [10, 20, 30, 40]
last = data.pop()
print(data)

[10, 20, 30]

values = [5, 10, 15]

3

True

List Iteration

c = ["New York", "London", "Tokyo"]
for city in c:
    print(city)

New York
London

List Comprehension

A = [x**2 for x in range(5)]
print(A)

[0, 1, 4, 9, 16]

A = [x**2 for x in range(10)]
print(A)

[0, 1, 4, 9, 16, 25, 36, 49, 64, 81]
