Basic if Statement

Using if else Statement

Number = 30
if Number < 20:
    print("Number is small")
else :
    print("Number is big")

Number is big

Using if & elif & else Chain

Mark = 89

if Mark >= 90:
    print("A")
elif Mark >= 75:
    print("B")
else:
    print("C")

B

Nested if Statements

user = "swathi"
pas = "1234"

if user == "swathi":
    if pas == "1234":
        print("Access")
    else:
        print("Incorrect")

Access

Using Logical Operators in Conditions

age = 25
verified = True

if age > 18 and verified:
    print("proceed")

proceed

Short-Hand if Statement 

x = 45
if x > 10: print("x is greater than 5")

x is greater than 5

Short-Hand if & else

num = 89

res = "Even" if num % 2 == 0 else "Odd"
print(res) 

Odd

Checking Multiple Values with in

day = "Monday"

if day in ["Saturday", "Sunday"]:
    print("Weekend")
else:
    print("Weekday")

Weekday

Comparing Strings in Conditions

user = "Swathi"

if user.lower() == "swathi":
    print("Welcome Swathi...")

Welcome Swathi...

value = 5

    pass
