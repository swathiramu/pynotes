Basic for Loop with a Sequence

fruits = ["apple", "banana", "orange"]

for fruit in fruits:
    print(fruit)

banana
orange

Custom Start and Step with range()

for i in range(2, 10, 3):
    print(i)

Iterating Through a String

name = "Swathi"

for char in name:
    print(char)

Using for with Index via enumerate()

lang = ["Python", "Java", "C++"]

for index, lan in enumerate(lang):
    print(index, lan)

0 Python
1 Java
2 C++

Nested for Loops

for i in range(4):
    for j in range(2):
        print(f"i={i}, j={j}")

i=0, j=0
i=0, j=1
i=1, j=0
i=1, j=1
i=2, j=0
i=2, j=1
i=3, j=0
i=3, j=1
